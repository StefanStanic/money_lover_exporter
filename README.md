# Money lover backup

Money lover backup exports all the data from your money lover app and stores it into CSV

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to setup the script.

```bash
pip install . 
```

## Usage

```bash
python money_lover_backup
```
